# Private Beta Gate Module

This module is pure static TypeScript data for the Visual Lexicon Track B
Private Beta Gate.

It records the #165 evidence source, the required zero-P0 threshold, the
private/manual gate-review verdict, the public paid beta No-Go verdict, owner
approval requirements, disabled payment/checkout/payment SDK flags, analytics
readiness events, support/refund/privacy checklist items, and stop conditions.

The module does not process payment, grant paid entitlement, add routes, read or
write browser storage, call network helpers, import auth or payment providers,
or mutate production state.

The canonical human-readable report is:

```txt
docs/PRIVATE_BETA_GATE.md
```
